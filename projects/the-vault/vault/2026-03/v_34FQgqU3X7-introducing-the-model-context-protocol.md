# Introducing the Model Context Protocol

> Anthropic is open-sourcing the Model Context Protocol (MCP), a new standard designed to connect AI assistants directly to various data sources and systems. This protocol aims to solve the problem of information silos by replacing fragmented integrations with a universal, open architecture for secure two-way connections. Early adopters like Block and Apollo are already integrating MCP to enable AI agents to retrieve relevant context more effectively. Developers can start building connectors today using pre-built servers or SDKs available through the Claude Desktop app.

## Key Points
- MCP is an open standard enabling secure, two-way connections between data sources and AI tools
- It replaces fragmented integrations with a universal protocol to help models access necessary data without isolation
- Pre-built servers for systems like Google Drive, Slack, and GitHub are available for immediate use via Claude Desktop
- Companies like Block and Apollo have already integrated MCP to enhance their agentic systems and workflows
- Developers can build custom connectors using SDKs and contribute to the open-source repository of implementations
