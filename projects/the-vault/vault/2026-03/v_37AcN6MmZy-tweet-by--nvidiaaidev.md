# Tweet by @nvidiaaidev

> NVIDIA has released OpenShell, an open-source runtime designed to make autonomous AI agents safer and easier to deploy. This tool acts as a security layer between agents and infrastructure, allowing developers to control execution, visibility, and inference without modifying existing code. By enabling fine-grained privacy and security controls, OpenShell addresses the critical infrastructure gap preventing widespread agent adoption. Ultimately, this allows any SaaS company to safely integrate coding or claw agents into their workflows.

## Key Points
- NVIDIA OpenShell is an open-source runtime for building with autonomous evolving agents
- It sits between the agent and infrastructure to govern execution, visibility, and inference location
- The tool provides fine-grained control over privacy and security while maintaining agent productivity
- Deployment requires only one command with zero code changes needed for existing agents
- Compatible agents include OpenClaw, Anthropic’s Claude Code, and OpenAI’s Codex
