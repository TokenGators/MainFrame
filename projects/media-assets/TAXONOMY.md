# Gatorpedia Tag Taxonomy v1.0

Tags are **flat, lowercase, hyphenated**. Assets can have multiple tags.

**Rules:**
- All AI tagging must draw exclusively from this list
- New tags require human approval before use — add proposals to `TAXONOMY-PROPOSALS.md`
- Tier 1 tags are set automatically from asset type — do not apply manually
- Tier 4 (tone) tags are provisional — all AI-assigned tone tags require human review

## Tier 4 — Open Descriptors

*Free-form descriptive tags generated by AI. No fixed list — think Giphy-style metadata: subjects, moods, themes, aesthetics, actions, settings, emotions. The goal is rich discoverability. Any number of tags is fine. These are NOT validated against a controlled vocabulary.*

**Guidance for AI:** Generate as many descriptors as genuinely apply. Cover:
- **Mood / tone:** celebratory, mysterious, tense, humorous, nostalgic, triumphant, melancholy, hype, chill, dark, surreal, wholesome, chaotic...
- **Visual aesthetic:** neon, glitch, retro, cinematic, pixel-art, hand-drawn, lo-fi, high-contrast, swamp, underwater, urban...
- **Subjects / characters:** alligator, swamp, crowd, city, fire, water, forest, robot, skull, food, sports, dance...
- **Actions / energy:** running, flying, celebrating, fighting, transforming, exploding, floating, spinning...
- **Themes:** friendship, loyalty, rebellion, power, mystery, survival, nostalgia, absurdity, parody, origin-story...
- **Setting / context:** nighttime, festival, arena, jungle, digital-world, outer-space, underwater, street...

Do NOT repeat Tier 2 or Tier 3 tags here. These descriptors are additive context.
Do NOT generate tags that duplicate the asset type (e.g. don't tag an NFT record with "nft" — the type field already captures that).

## Adding New Tags

1. Add your proposed tag to `TAXONOMY-PROPOSALS.md` with a justification
2. Human approves → move to this file under the appropriate tier
3. Once in this file, the tag is available for AI tagging jobs

Do **not** add tags directly to this file without the proposal process.
