# Gatorpedia

**Type:** media-library | digital-assets
**Stack:** Static media assets (images, video, audio)
**Status:** active
**Deploy:** Not deployed (static assets)

## Overview
Centralized repository for TokenGators brand media, event announcements, promotional materials, and digital assets.

## Contents
- `assets/images/` — Event teasers, announcements, promotional images
- `assets/videos/` — Event footage, announcements, promotional videos
- `assets/audio/` — Audio files, music, sound effects

## Asset Organization
Assets are named with descriptive labels and dates (YYYYMMDD format):
- event-teaser-20260304.jpg
- gm-announce-20260304.jpg
- floor-chart-20260304.png
- return-to-swamp-live-now-in-othersidemeta-...mp4

## Usage
- Reference in game projects, web apps, marketing materials
- Games can import images/audio for UI, menus, marketing assets
- Maintain consistent naming and dating conventions

## Agent instructions
- Organize assets by type (images, video, audio)
- Use descriptive names + date stamps for new assets
